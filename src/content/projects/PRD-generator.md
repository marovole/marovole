## 为什么做这个

作为产品经理，我发现自己经常花大量时间在重复的 PRD 写作上。这个工具帮助我快速从想法到结构化的需求文档。

## 核心特性

### 智能大纲生成
输入产品想法，AI 自动生成完整的 PRD 大纲结构。

### 需求细化
针对每个功能点，AI 会提出关键问题和潜在风险。

### 导出多格式
支持导出为 Markdown、Notion、Figma 等多种格式。

## 使用场景

1. **快速头脑风暴**: 输入想法，快速获得结构化的大纲
2. **PRD 模板**: 为常见产品类型提供定制化模板
3. **需求评审**: 生成的需求文档可直接用于团队评审

## 技术栈

- **前端**: Astro + Tailwind CSS
- **AI**: Claude API
- **部署**: Cloudflare Pages
