## 核心功能

- **深度研究**: Glass Box UX 透明化研究过程，实时展示 AI 思考链
- **红牌仪表盘**: 风险预警，自动识别项目潜在风险
- **诈骗风险评估**: 基于多维度指标的 ScamMeter
- **持币分布分析**: 识别巨鲸和集中度风险
- **代币解锁日历**: 追踪 vesting 计划
- **实时价格数据**: 多轮对话上下文保持

## 技术栈

- **前端**: React 18 + TypeScript + Vite + TailwindCSS + Radix UI
- **后端**: Cloudflare Workers + Hono + TypeScript
- **数据库**: Supabase PostgreSQL
- **AI**: OpenRouter 多模型路由 (DeepSeek, GPT, Tongyi)
- **搜索**: Brave Search, Tavily, Serper
